Privacy Policy

Last updated: March 2026

1. Overview

This application is a personal automation tool. It is not a public-facing product and does not serve end users. This privacy policy explains how the tool handles any data it interacts with.

2. Data collection

This application does not collect, store, or process personal data from any users other than the developer's own TikTok account credentials, which are stored locally and never shared.

3. TikTok API data

This application accesses the TikTok API solely to post video content to the developer's own account. No user data is extracted, stored, or transmitted to any third party.

4. Cookies

This application does not use cookies or any tracking technologies.

5. Data sharing

No data is shared with any third parties. All API credentials are stored locally in environment variables on the developer's own machine or private server.

6. Contact

For any questions regarding this privacy policy, please reach out via the associated GitHub repository.
